# Исправление ошибки перелистывания недель в календаре

## Проблема
При перелистывании недель в личном кабинете учителя происходил пропуск недель. Например, при переходе с недели 18-24 сразу открывалась неделя 1-7, пропуская неделю 25-31.

## Причина
Ошибка была в функции `getWeekStart()` в файле `dashboard.js` и аналогичной логике в `admin-lessons.js`. 

### Неправильная логика:
```javascript
if (dayOfWeek === 0) { // Воскресенье
    daysToMonday = 1; // До следующего понедельника
} else {
    daysToMonday = dayOfWeek - 1; // До понедельника этой недели
}
```

Когда пользователь находился в воскресенье, код переходил к следующему понедельнику вместо понедельника текущей недели.

## Исправление
Изменил логику вычисления начала недели:

### Исправленная логика:
```javascript
if (dayOfWeek === 0) { // Воскресенье
    daysToMonday = 6; // До понедельника этой недели (6 дней назад)
} else {
    daysToMonday = dayOfWeek - 1; // До понедельника этой недели
}
```

## Затронутые файлы
1. `brainify/src/main/resources/static/js/dashboard.js` - функция `getWeekStart()`
2. `brainify/src/main/resources/static/js/admin-lessons.js` - функция `updateWeekDisplay()`

## Результат
Теперь перелистывание недель работает корректно:
- 18-24 → 25-31 → 1-7 (следующий месяц)
- Нет пропусков недель
- Правильное вычисление начала недели для любого дня недели
